# Vespa Overlay
Vespa Gentoo Overlay

## How to add to your Gentoo:

```bash
sudo eselect repository add vespa git https://github.com/thiagovespa/vespa-overlay.git
```

To sync:
```bash
sudo emaint sync -r vespa
```
